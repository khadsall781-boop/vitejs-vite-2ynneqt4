# Tornado Tacklers' Map

A real-time GPS tracking application for storm chasers. This tool allows teams to coordinate their movements, share live locations, and monitor weather events collectively.

## Features

- **Real-time Tracking**: Live GPS updates for all active team members.
- **Interactive Map**: Built with Leaflet for smooth navigation and marker management.
- **Chaser Management**: Secure dashboard for adding and updating chaser profiles.
- **Live Status**: See who is currently active and their last reported location.
- **Supabase Integration**: Robust backend for authentication and real-time data syncing.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: CSS3 (Custom properties for theming)
- **Mapping**: Leaflet
- **Backend/Auth**: Supabase

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your Supabase environment variables in a `.env` file.
4. Run the development server: `npm run dev`
